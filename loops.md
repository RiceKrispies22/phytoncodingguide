# For Loops
A for loop iterates over each item in a sequence, such as a list, string, or range. The loop runs once for each item

```python
fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)
```
or

```python
word = "hello"

for letter in word:
    print(letter)
```

# While Loops

A while loop runs as long as a condition stays true. When the condition becomes false, the loop stops.


```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

or

```python
number = 5

while number > 0:
    print(number)
    number -= 1

print("Blast off!")
```


# range
A range creates a sequence of numbers. It is mainly used with for loops.

```python
for i in range(5):
    print(i)
```

or

```python
for i in range(2, 6):
    print(i)
```
