# If Statement
An if statement runs code only if a condition is true. Only if its true.

```python
age = 16

if age >= 16:
    print("you are able to drive")
```

# If / Else
If/Else means the code only runs if the condition is false.
age = 12

```python
if age >= 16:
    print("you can drive the car")
else:
    print("ur too young gng")
```

# If / Elif / Else
elif means “else if” and it lets you check multiple conditions.

```python
score = 67

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: D")
```
